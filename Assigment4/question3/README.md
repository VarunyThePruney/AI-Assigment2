# Sudoku CSP Solver

## Overview

This project implements a **Sudoku solver using Constraint Satisfaction Problem (CSP)** with **Backtracking in Python**.
The program fills the missing cells in a 9×9 Sudoku grid while satisfying Sudoku constraints.

## Constraints

* Each number (1–9) appears once in every row
* Each number (1–9) appears once in every column
* Each number (1–9) appears once in every 3×3 grid

## How to Run

```bash
python sudoku.py
```

## Output

The solved Sudoku grid is printed in the terminal.

## Algorithm

Backtracking CSP
